## 📊 Data Flow

```
┌─────────────┐
│   USER      │
└──────┬──────┘
       │
       │ Clicks "Shop All" / Applies Filter / Searches
       │
       ▼
┌─────────────────┐
│  ProductList.jsx│
└────────┬────────┘
         │
         │ Fetches: GET /api/products?filters...
         │
         ▼
┌────────────────────┐
│  Backend API       │
│  productRoutes.js  │
└────────┬───────────┘
         │
         │ Queries MongoDB with filters
         │
         ▼
┌────────────────────┐
│  Product Model     │
│  (MongoDB)         │
└────────┬───────────┘
         │
         │ Returns: { products, pagination, filters }
         │
         ▼
┌─────────────────┐
│  ProductCard    │ (Maps products)
└────────┬────────┘
         │
         │ User clicks product
         │
         ▼
┌─────────────────────┐
│  ProductDetails.jsx │
└────────┬────────────┘
         │
         │ Fetches: GET /api/products/:slug
         │
         ▼
┌────────────────────┐
│  Single Product    │
│  (Full Details)    │
└────────────────────┘
```

## 🔧 Configuration Options

### ProductList.jsx
```javascript
pagination: {
  limit: 12,          // Products per page
  page: 1             // Current page
}

viewMode: 'grid'      // 'grid' or 'list'
sortBy: 'newest'      // Default sort
```

### FilterSidebar.jsx
```javascript
priceRange: {
  min: 0,
  max: 10000          // Default max price
}

openSections: {       // Accordion states
  category: true,
  price: true,
  colors: true,
  material: true
}
```

### ProductCard.jsx
```javascript
imageHover: true      // Show 2nd image on hover
badgePosition: 'top-left'
wishlistPosition: 'top-right'
```
